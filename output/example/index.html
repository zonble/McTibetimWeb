McTibetim - Online Tibetan Input Method

API Reference

Keyboard Layout

McTibetim is an input method for typing Tibetan script built with web technologies. Besides the web version, it also provides a Chrome extension for inputting Tibetan script on Chrome OS.

OpenVanilla Web Input Methods
  • McBopomofo (小麥注音)
  • McFox (小麥族語)
  • McTabim (小麥他命)
